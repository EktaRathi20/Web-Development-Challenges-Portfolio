# Web-Development-Challenges-Portfolio
#### Project Overview
Welcome to my Web Development Challenges Showcase! In this repository, I've compiled a series of projects I worked on during my early days of learning HTML, CSS, and JavaScript. Each folder represents a specific challenge where I explored various aspects of frontend development.

#### Challenges Overview

1. **Challenge 1 - Sign Up, Login, Home Screen, Payment Form, Survey Form (Basic HTML CSS)**
    - Description: A set of basic web pages covering essential form elements and layouts.

2. **Challenge 2 - Gym (HTML CSS)**
    - Description: A simple webpage focused on building a gym-related interface with HTML and CSS.

3. **Challenge 3 - Float Property (HTML CSS)**
    - Description: Exploration of the float property in CSS for layout design.

4. **Challenge 4 - Position Property (HTML CSS)**
    - Description: Understanding and implementing the position property in CSS for element positioning.

5. **Challenge 5 - Flex Property (HTML CSS)**
    - Description: Creating flexible and responsive layouts using the flexbox model in CSS.

6. **Challenge 6 - Grid Property (HTML CSS)**
    - Description: Introduction to CSS grid for building grid-based layouts.

7. **Challenge 7 - Animation Property (HTML CSS)**
    - Description: Implementing basic animations using CSS animation properties.

8. **Challenge 8 - Animated Login and Sign Up Form (HTML CSS JS)**
    - Description: Building an animated login and sign-up form with HTML, CSS, and JavaScript.
